Currency Converter Android App
This project is a simple Android currency converter developed as part of a coursework assignment.
The application allows users to enter an amount in AUD and convert it to USD, EUR, or CNY. The conversion result is updated immediately based on the selected currency.

Main Features
Input an amount in Australian Dollars (AUD)
Convert AUD to USD, EUR, or CNY
Display conversion results with basic input validation
User interface implemented using Jetpack Compose and Material Design
Implementation
The app is written in Kotlin and uses the modern Android API.
Jetpack Compose is used to build the UI in a declarative way.
Exchange rates are defined locally in the application.
State is managed using Compose state functions such as `remember` and `mutableStateOf`.
