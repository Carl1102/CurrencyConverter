# Currency Converter Android App

This is a simple Android currency converter developed as part of a coursework assignment.

The app allows users to enter an amount in Australian Dollars (AUD) and convert it to USD, EUR, or CNY.  
The conversion result updates immediately when the input value or target currency changes.

## Features
- Input amount in AUD
- Convert AUD to USD, EUR, or CNY
- Basic input validation for numeric values
- Clean UI built with Jetpack Compose and Material Design

## Implementation Details
- The application is written in Kotlin using modern Android APIs.
- Jetpack Compose is used to build the user interface in a declarative way.
- Exchange rates are defined locally within the app.
- UI state is managed using Compose state utilities such as `remember` and `mutableStateOf`.

## Purpose
This project was created to practise Android development concepts, including:
- Compose-based UI design
- State management
- Basic application structure and navigation
